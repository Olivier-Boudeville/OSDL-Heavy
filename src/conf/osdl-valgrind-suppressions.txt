# This is the OSDL suppression file, to be used with Valgrind.

# Comments:
#
#   - we kept from the call stack only the parts in third-party libraries
# (i.e. the initial Ceylan and OSDL calls were removed)
#
#	- depending on a 32-bit or 64-bit OS being used, we had for example either
# obj:/usr/lib/libX11.so.6.3.0 or /usr/lib/x86_64-linux-gnu/libX11.so.6.3.0
# (thus we retained first obj:/usr/lib/*libX11.so.6.3.0)
#
#   - we do not want suppressions to depend on library versions, so lines
# including:
# obj:/usr/lib/*libX11.so.6.3.0 were transformed in obj:/usr/lib/*libX11.so.*
# obj:/usr/lib/*libXext.so.6.4.0 were transformed in obj:/usr/lib/*libXext.so.*
# obj:/usr/lib/*libstdc++.so.6.0.13 were transformed in
# obj:/usr/lib/libstdc++.so.*
# obj:/lib/ld-2.11.1.so were transformed in obj:/lib/ld-*.so
# obj:/usr/lib/libxcb.so.1.1.0 were transformed in obj:/usr/lib/*libxcb.so.*
# obj:/usr/lib/nvidia-current/libGL.so.270.41.06 transformed in
# obj:/usr/lib/nvidia-current/libGL.so.*
# obj $PREFIX/Ceylan-0.7/lib/libCeylan-0.7.so.0.0.7 were transformed in
# */Ceylan-*/lib/libCeylan-*.so.*
# obj:/usr/lib/libpulsecommon-0.9.22.so were transformed in
# obj:/usr/lib/libpulsecommon-*.so
# obj:/usr/lib/libpulse.so.0.12.3 were transformed in obj:/usr/lib/libpulse.so.*
# etc.; same process for @@GLIBC_*.


# Sections:
#
# 1. SDL/X11 related suppressions
# 2. SDL_image related suppressions
# 3: audio-related suppressions (SDL, libpulse, etc.)
# 4. other SDL related suppressions
# 5. PhysicsFS related suppressions
# 6. CEGUI related suppressions
# 7. Freetype related suppressions
# 8. OSDL related suppressions
# 9. others suppressions

# (Ceylan suppressions listed in ceylan-valgrind-suppressions.txt)
