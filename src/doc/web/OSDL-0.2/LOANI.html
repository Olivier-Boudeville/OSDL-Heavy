LOANI's corner

Overview

This is the manual page for LOANI version 0.2.

LOANI stands for Lazy OSDL's Automatic Net Installer. Its purpose is to simplify as much as possible the installation of OSDL and its pre requesites. By Automatic, one should understand "as automatic as feasible in not too long a time". Anyway, some user interaction is necessary (for example, where should LOANI put the resulting binaries ?).

LOANI pre requesites

You of course should have a working internet connection in order to retrieve the needed packages.

Necessary tools (GNU version strongly recommended)

  • bash
  • cvs (command-line client)
  • wget
  • configure
  • make
  • tar
  • a good ISO/C++ compiler, such as gcc, in a not too old version (no 2.96 version for example, prefer 3.3 and above)

Optional

  • dialog
  • whiptail

(please note that these pre requesites are LOANI's ones, a subset of full OSDL's ones).

What LOANI provides

LOANI is a simple bash script which automatically retrieves, configures, compiles, installs and links together the following libraries :

  • SDL
  • Ceylan
  • OSDL


How to get it and use it

You should download latest version of LOANI from CVS (choose the latest revision and click download). Then you can enter :

bash loani.sh


Answer the questions and everything should go fine. If ever errors occured (which is far from impossible), one may take a look at loani.sh (this simple script is easy to understand and customize) and/or send us a notice to have LOANI improved.
