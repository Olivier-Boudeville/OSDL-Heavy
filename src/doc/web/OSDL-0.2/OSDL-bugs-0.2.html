OSDL's Bug parade

Infamous ! Outrageous ! OSDL (or Ceylan) would have bugs ?!? We do our best to pursue and eliminate them, and you can help us in this never-ending quest by kindly submitting the ones you might discover.

Please look at our SourceForge's tracker tool to get an insight and, if needed, report a bug.

Otherwise, you can send us a bug report form. We have not yet formalized it for the moment, please explain everything which you think related to the problem, we will do our best to correct it.
