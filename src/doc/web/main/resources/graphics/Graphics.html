Graphical content repository

Hereby are listed all our known usable sources for graphical content (2D : sprites and textures, 3D : models, etc.)

Thanks to mods and tools like POV, many talentuous artists allow their 2D drawings, 3D models and textures to be used by others : characters, objects, vehicles, buildings, places, etc. are available in full artwork repositories.

One can state a trend in the artistic field which follows closely the open-source philosophy, which arose at first on the technical side. That's a promising change !
