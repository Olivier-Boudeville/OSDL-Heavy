La correction gamma

Principe

La correction gamma, ou gamma-correction, consiste à appliquer un filtre pendant le rendu des pixels de telle sorte que leur luminosité soit modifiée [Plus d'infos, en anglais].

L'usage premier est de compenser les distorsions de luminosité induites par les dispositifs effectifs de rendu, notamment les écrans : si l'on veut afficher [R;G;B] sur un dispositif qui aurait affiché dans les faits [Gr(R);Gg(G);Gb(B)], il suffit de vouloir afficher les valeurs des transformations inverses [Gr^-1(R);Gg^-1(G);Gb^-1(B)] pour que in fine les couleurs restituées soient correctes.

La correction gamma peut aussi être employée pour effectuer des fondus au noir ou entre deux images.

SDL permet de gérer la correction gamma à l'aide de :

  • SDL_SetGammaRamp
  • SDL_SetGamma
