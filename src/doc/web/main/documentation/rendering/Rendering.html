Quick overview

The purpose of this section is to give details about the way rendering (both for graphics and sound) is achieved. There are some articles written in French, and our SDL corner.

Quelques informations sur le rendu graphique

Présentation

Ce document recense quelques enseignements portant sur les méthodes de rendu graphique. Ces informations nous paraissent rarement précisées ou correctement expliquées, de telle sorte qu'il nous est pratique d'en disposer directement pour notre projet.

Actuellement, nous nous sommes concentrés sur les notions les plus élémentaires, en nous restreignant au rendu 2D. Les explications concernant l'application de ces concepts se réfèrent aux conventions SDL, la couche de portabilité bas niveau que notre bibliothèque OSDL exploite.
