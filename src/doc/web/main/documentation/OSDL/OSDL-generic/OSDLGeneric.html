OSDL's generic information

Overview

This part of OSDL's documentation focuses on pieces of information which are not directly relevant to a specific version of our library.

Notably are discussed here some design choices regarding architecture (in French), what is our quality policy, some general practical informations (about the languages being used, how to use SVN and prepare new OSDL versions, etc.). Some similar projects are detailed as well.
