OSDL's generic informations

Overview

This part of OSDL's documentation focuses on all informations which are not directly relevant to a specific version of our library.

Notably are discussed here some design choices regarding architecture (in french), what is our quality policy, some general practical informations (about the languages being used, how to use CVS and prepare new OSDL versions). Some similar projects are detailed as well.
