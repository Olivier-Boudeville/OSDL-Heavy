Homepage of OSDL's 0.5 version

Overview

As of March 2007, this is the current testing OSDL's version, and it is still work on progress (not yet officially released, hence beta version). Several features are however already available and thoroughfully tested.

For the moment, OSDL 0.5 is thoroughly tested only on GNU/Linux hosts [More infos]. This OSDL version uses the Ceylan 0.7 version.

If you are searching for general informations about the OSDL library (not specific to the 0.5 version), prefer referring to our OSDL official website.

Main changes

OSDL license has been extended and updated, went from pure LGPL to disjunctive bi-license LGPL/GPL. It allows developers using GPL software to use OSDL as well.
