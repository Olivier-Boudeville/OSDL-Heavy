Homepage of OSDL's 0.1 version

Overview

This was the first version of the OSDL library. Its only merit was that it existed. This version does not provide any particular functionality : we moved from the 0.1 version to the 0.2 only because we screwed up the CVS repository of version 0.1, being unable to delete permanently old directories.

This version is merely only available for historical reasons, so that people cannot ask us why the 0.1 version is nowhere to be found !
