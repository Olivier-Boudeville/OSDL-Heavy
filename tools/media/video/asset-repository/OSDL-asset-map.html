Overview

This page allows to keep track of the various assets and their transformations, so that they can be played back if needed.

Video Assets

Regarding animations coming from Reiner's Tilesets, see also these scripts:

  1. retrieve-reiner-content.sh, to download a local copy of the animations
  2. sort-reiner-mirror.sh, to sort animations by theme (see the result)
  3. process-reiner-content.sh, to trigger the full processing of all Reiner content
  4. process-reiner-individual-archive.sh, to process a particular archive from Reiner content (used by process-reiner-content.sh)

Animated objects

Their name is the human-readable identifier defined in animated-object-identifiers.txt. Same approach for outside looks (outside-looks-identifiers.txt), for attitudes (attitude-identifiers.txt) and directions (direction-identifiers.txt).

Instances in each category (object, outside look, etc.) are listed in the order defined in their repository (i.e. alphabetical order).

For each animated object, all available outside looks are listed and, for each outside look, all available attitudes are listed (in identifier order) between brackets. Unless specified otherwise, an attitude is available for all eight directions. Directions are not listed in alphabetical order: they are listed so that a direction corresponds to the previous one once rotated clockwise of 1/8 turn (i.e. 45 degrees), starting from North.
